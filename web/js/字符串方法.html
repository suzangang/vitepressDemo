字符串方法 ​

split() ​

将字符串以指定分隔符拆分成数组。 ​

javascript
let str = "apple, banana, cherry";
let arr = str.split(","); // 将字符串按逗号分割成数组
console.log(arr); // ["apple","banana","cherry"]

slice() ​

截取字符串并返回新的字符串。 ​

javascript
let str = "Hello, World!";
let substr = str.slice(0, 5); // 从索引0开始，截取到索引5之前的字符
console.log(substr); // "Hello"

indexOf() ​

返回指定字符串在原字符串中第一次出现的索引位置。 ​

javascript
let str = "Hello, World!";
let index = str.indexOf("o");
console.log(index); // 4

lastIndexOf() ​

返回指定字符串在原字符串中最后一次出现的索引位置。 ​

javascript
let str = "Hello, World!";
let index = str.lastIndexOf("o");
console.log(index); // 8

replace() ​

替换字符串中的指定子串。 ​

javascript
let str = "Hello, World!";
let newStr = str.replace("World", "JavaScript");
console.log(newStr); // "Hello, JavaScript!"

trim() ​

去除字符串两端的空白字符。 ​

javascript
let str = "   Hello, World!   ";
let trimmedStr = str.trim();
console.log(trimmedStr); // "Hello, World!"

toUpperCase() ​

将字符串转换为大写。 ​

javascript
let str = "Hello, World!";
let upperCaseStr = str.toUpperCase();
console.log(upperCaseStr); // "HELLO, WORLD!"

toLowerCase() ​

将字符串转换为小写。 ​

javascript
let str = "Hello, World!";
let lowerCaseStr = str.toLowerCase();
console.log(lowerCaseStr); // "hello, world!"
